# BrowserBible v4

Bible software that runs in the browser, created by [John Dyer](https://j.hn/)
and maintained by the Digital Bible Society.

### Getting Started

Download this [collection](https://bibles.dbs.org/_assets/starter-pack.zip) of
17 freely shareable bibles with only 95Mb they cover about ~90% of the world's
population. For additional bibles please check out [dbs.org](https://dbs.org),
[ebible.org](https://ebible.org/find/) and [fetch.bible](https://fetch.bible).

```
wget https://bibles.dbs.org/_assets/starter-pack.zip && unzip starter-pack.zip
```
